1. head tag

    2. The head element is a container for metadata and is placed between the html tag and the body tag. Metadata is data about the HTML document.

  2. title tag

    3. The title tag is an HTML code tag that allows you to give a web page a title.

  3. a tag

    4. The tag defines a hyperlink, which is used to link from one page to another.

  4. div tag

    5. The HTML element is the generic container for flow content. It has no effect on the content or layout until styled in some way using CSS.
